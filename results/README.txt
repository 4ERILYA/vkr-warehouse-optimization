Эта папка заполняется автоматически при запуске run_experiment.py.

Содержит:
  summary.csv             — сводная таблица сравнения политик
  all_runs.csv            — детальные данные по всем 120 прогонам (4 политики × 30)
  skus_with_params.csv    — таблица SKU с рассчитанными SS, ROP, EOQ, CSL
  comparison.png          — Парето-диаграмма политик (запас vs Fill Rate)
  metrics_bars.png        — Bar-charts по Fill Rate и среднему запасу
  cost_decomposition.png  — стек-bar декомпозиции издержек
  groups_effect.png       — эффект P2 над P1 по группам ABC-XYZ
  inventory_dynamics.png  — динамика запаса для одной SKU (s,Q-пила)
